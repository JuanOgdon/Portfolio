# Juan Ignacio Ogdon — Engineering Portfolio

I'm a 6th-year Aerospace Engineering student at Universidad Tecnológica Nacional (UTN), Buenos Aires, graduating December 2026. I focus on **control systems**, **genetic algorithms**, **neural network-based controllers**, and **autonomous aerospace applications**.

My work spans the full stack of an aerospace control problem: from physical modeling and simulation, through algorithm and controller design, to implementation and testing. I'm drawn to problems where the solution lives in the design of the method — not just the choice of a library — and I consistently build the tools I need from scratch when necessary.
